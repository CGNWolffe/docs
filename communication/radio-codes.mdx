Radio codes keep transmissions short, clear, and consistent across all units. Use this page as your go-to reference when preparing for patrol or when a code comes over the air that you don't immediately recognize.

<Info>
  Memorize the codes you'll use every shift — 10-4, 10-7, 10-8, 10-20, and Code 3 especially. Keep this page bookmarked for anything less common.
</Info>

## Ten-codes

Ten-codes (also called 10-codes) are the standard shorthand for OPP GLJRP radio communications. Every transmission should use the appropriate ten-code rather than spelling out the full phrase.

| Code | Meaning | Usage example |
|------|---------|---------------|
| 10-1 | Receiving poorly / signal weak | "Dispatch, you're 10-1, please repeat." |
| 10-2 | Receiving clearly / signal good | "10-2, copy that." |
| 10-4 | Acknowledged / understood | "10-4, en route." |
| 10-6 | Busy — stand by | "10-6, finishing a stop." |
| 10-7 | Out of service | "Unit 42, 10-7 for end of shift." |
| 10-8 | In service / available | "Unit 42, 10-8, clear from scene." |
| 10-9 | Repeat last transmission | "10-9, did not copy." |
| 10-10 | Fight in progress | "10-10 reported at Sandy Beach Road." |
| 10-11 | Animal complaint | "10-11 on Portage Road." |
| 10-15 | Have suspect in custody | "Unit 12, 10-15, one in custody." |
| 10-20 | Location / position | "What's your 10-20?" — "Highway 20 near Exit 4." |
| 10-21 | Call by phone | "10-21 the complainant." |
| 10-22 | Disregard / cancel | "10-22 that last call, situation resolved." |
| 10-23 | Arrived at scene | "Unit 7, 10-23." |
| 10-25 | Report to / meet with | "10-25 Sergeant at the detachment." |
| 10-27 | Driver's licence check | "Running a 10-27 on the registered owner." |
| 10-28 | Vehicle registration check | "10-28 on Ontario plate ABCD 123." |
| 10-29 | Check for warrants | "Running 10-29 on the subject." |
| 10-33 | Emergency — all units clear the air | "10-33, 10-33 — officer needs assistance." |
| 10-35 | Confidential information / major crime alert | "10-35, switch to secondary channel." |
| 10-42 | Officer's end of shift / off duty | "Unit 42, 10-42, good night." |
| 10-99 | Officer in danger / urgent assistance needed | "10-99! Shots fired, I need backup now!" |

<Warning>
  **10-33 and 10-99 are emergency codes.** Do not use them unless there is a genuine emergency. When either code is broadcast, all non-emergency traffic must immediately clear the channel.
</Warning>

## Signal codes

Signal codes supplement ten-codes and are used by dispatch to categorize calls before assignment.

| Signal | Meaning |
|--------|---------|
| Signal 0 | Deceased person |
| Signal 1 | Motor vehicle collision (no injuries) |
| Signal 2 | Motor vehicle collision (with injuries) |
| Signal 3 | Motor vehicle collision (fatality) |
| Signal 4 | Fire — structure |
| Signal 5 | Fire — vehicle |
| Signal 6 | Domestic disturbance |
| Signal 7 | Suspicious person or vehicle |
| Signal 8 | Break and enter in progress |
| Signal 9 | Robbery in progress |
| Signal 10 | Assault in progress |

<Note>
  Dispatch will typically broadcast the signal code along with the address and any available details. Acknowledge with your unit number and 10-4, then respond with your estimated arrival or advise if you're unable to respond.
</Note>

## Response priority levels

Every call is assigned one of three priority levels. Your driving response and conduct on scene depend on the priority designation.

<AccordionGroup>
  <Accordion title="Code 1 — Routine response" defaultOpen={false}>
    **No lights or sirens.** Proceed at normal road speed following all traffic laws.

    Code 1 applies to non-urgent calls where there is no immediate risk to life or property — for example, a parking complaint, a routine follow-up, or a report of a past occurrence. Take your time, drive safely, and arrive composed.
  </Accordion>

  <Accordion title="Code 2 — Urgent response" defaultOpen={false}>
    **Lights only, no siren** unless traffic conditions require it.

    Code 2 applies to calls that are time-sensitive but do not yet meet the threshold for emergency driving — for example, a disturbance that is still active but not involving weapons, or a suspicious vehicle that may leave the area. Drive with urgency but within reason.
  </Accordion>

  <Accordion title="Code 3 — Emergency response" defaultOpen={false}>
    **Full lights and siren.** Emergency vehicle operation is authorized.

    Code 3 applies to calls where there is an immediate risk to life — an assault in progress, an officer down, a vehicle pursuit, or a confirmed 10-99. Follow all detachment vehicle pursuit and emergency driving policies. Notify dispatch that you are responding Code 3.

    **Example radio transmission:** *"Dispatch, Unit 42 responding Code 3 to that 10-10 on Lundy's Lane."*
  </Accordion>
</AccordionGroup>

## How to use codes during radio communications

Follow this structure every time you key up your radio:

<Steps>
  <Step title="Identify the station you are calling">
    Always address dispatch or the specific unit first: *"Dispatch, Unit 42..."* or *"Unit 12, Unit 42..."*
  </Step>
  <Step title="Wait for acknowledgement">
    Pause after the address. Dispatch will respond with your unit number when the channel is clear: *"Unit 42, go ahead."*
  </Step>
  <Step title="Deliver your message using codes">
    State your message concisely. Use ten-codes and signal codes where applicable. Avoid lengthy explanations over the air — switch to phone (10-21) for detailed information.
  </Step>
  <Step title="End your transmission">
    Conclude with your status code if it has changed. For example, after arriving: *"Unit 42, 10-23."* After clearing: *"Unit 42, 10-8."*
  </Step>
</Steps>

<Tip>
  Keep one hand free and think through your message before you key the radio. Dead air wastes channel time and can block emergency traffic.
</Tip>

## Common phrases and their radio equivalents

Use these radio equivalents instead of plain language wherever possible.

| Plain language | Radio equivalent |
|----------------|-----------------|
| "I understand / got it" | "10-4" |
| "Where are you?" | "What's your 10-20?" |
| "Cancel that call" | "10-22" |
| "I'm done for the day" | "10-7" or "10-42" |
| "I'm available" | "10-8" |
| "Say that again" | "10-9" |
| "I'm on scene" | "10-23" |
| "I have someone in custody" | "10-15" |
| "I need help immediately" | "10-99" |
| "Check this plate / licence" | "10-28" / "10-27" |
| "Responding with lights and siren" | "Responding Code 3" |
| "All units hold radio traffic" | "10-33" |
